## 项目简介

LocalDetector Pro 是一款专注于本地代码库安全审计与敏感信息防泄露的静态扫描工具。针对传统正则扫描器误报率较高、难以识别非结构化加密凭证的问题，本项目设计并实现了“正则表达式 + 香农信息熵 + YARA 规则”三重混合检测引擎。结合并发扫描架构与图形化用户界面，本项目能够高效对本地代码进行深度安全分析，并生成标准化的审计报告。

## 核心技术特性

* **三重混合检测引擎**：
    * **正则表达式引擎**：基于预设模式匹配硬编码的 IPv4/IPv6、手机号、邮箱地址、云服务鉴权密钥（AK/SK）及常见代码凭证关键字。
    * **启发式信息熵引擎**：采用香农信息熵（Shannon Entropy）算法动态计算字符串的复杂度，能够识别缺乏固定特征格式的高强度加密密钥（如 JWT Secret、高频 Base64 Token 等）。
    * **YARA 规则引擎**：底层集成业界标准的 YARA 引擎，支持通过二进制特征与复杂逻辑规则，捕获具有特定结构的代码后门、非标准私钥及其他潜在的威胁指标（IOCs）。
* **并发扫描架构**：系统根据运行环境动态分配线程池，规避单线程 I/O 瓶颈，有效提升对大规模文件目录的遍历、读取与匹配效率。
* **现代化交互界面**：基于 CustomTkinter 构建 GUI 面板，支持实时滚动显示扫描日志、进度条更新及多线程防阻塞交互。
* **审计报告导出**：支持将扫描结果一键导出为格式化的 HTML 或 CSV 报告，报告内含风险等级分类、引擎类型标识与触发规则的具体代码切片，便于安全人员快速复现与溯源。

## 环境依赖与安装

运行本项目需要 Python 3.8 或更高版本。

**1. 获取项目代码**
请确保完整获取项目目录结构，特别是存放特征规则的 `rules` 文件夹不可缺失。

**2. 安装依赖库**
在项目根目录（含有 `requirements.txt` 的目录）下，通过终端执行以下命令安装项目所需的第三方依赖：
```bash
pip install -r requirements.txt
```

*(注：Windows 环境下，`yara-python` 库通常包含预编译的 C 扩展，直接通过 pip 安装即可正常使用。)*

## 使用说明

1. 在项目根目录下，通过命令行启动主程序：

Bash

```
python main.py
```

1. 在弹出的用户界面中，点击“浏览目录”按钮，选择需要审计的本地代码文件夹。
2. 点击“启动深度审计引擎”开始分析，界面底部的日志窗口将实时输出扫描过程与匹配命中的风险项。
3. 扫描任务结束后，点击“导出报告”按钮，可选择将分析结果保存为易于阅读的 HTML 文件或便于数据处理的 CSV 文件。

## 目录结构说明

Plaintext

```
LocalDetector_Pro/
├── core/
│   ├── scanner.py       # 扫描引擎核心逻辑 (并发调度、正则/熵/YARA引擎封装)
├── ui/
│   ├── app.py           # GUI 界面渲染与 HTML 报告生成模块
├── rules/
│   ├── *.yar            # YARA 特征规则库文件 (核心检测规则配置)
├── main.py              # 程序启动主入口
├── requirements.txt     # 项目依赖声明文件
└── README.md            # 项目说明文档
```
